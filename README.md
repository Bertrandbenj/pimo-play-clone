# Welcome to NINŎ

## Project info

NINŎ is an anonymisation, data generation and pseudonymisation tool box, based on PIMO, LINO, MIMO and GISO.
NINŎ allow you to manipulate configuration files meant to facilitate the anonymisation process. 

It is composed of :
- a yaml editor with an input/output test section
- a graph of database schema
- a graph of ansible execution role
- a list of workspace file
- lovable developers


# Run with:

```sh
# Step 1:  Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
